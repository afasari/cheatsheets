## Quick Tips

### Navigation
- Use the **sidebar** on the left to browse by category
- Use the **search** feature (Ctrl+K or Cmd+K) to find specific commands
- Use the **navbar** to quickly access different sections

### Tips for Learning
1. **Practice commands** in a safe environment before using in production
2. **Build muscle memory** by using commands regularly
3. **Combine tools** to learn how different DevOps tools work together
4. **Bookmark favorites** for quick reference
5. **Check for updates** as tools and technologies evolve

### Using This Site

#### Dark Mode
Toggle between light and dark themes using the theme button in the navbar.

#### Search
Press `Ctrl+K` (Windows/Linux) or `Cmd+K` (Mac) to open the search dialog and search across all cheat sheets.

#### Edit on GitHub
Click the "Edit this page on GitHub" link at the bottom of any page to suggest improvements.
